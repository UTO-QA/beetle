# beetle
This a user-friendly utility with beautiful GUI to run comprehensive automation over any website.

Installation:

1) Have java JDK(1.6+) installed in your system.
2) Install any Java-IDE (preferably Java J2EE) (https://eclipse.org/downloads/)
3) Install Maven in Java J2EE. (http://eclipse.org/m2e/documentation/m2e-extension-development.html)
4) Setup the project my importing the repo using git. (http://wiki.eclipse.org/EGit/User_Guide#Starting_from_existing_Git_Repositories)
5) replace the sanitized code (code starting and ending with ***)
6) compile the code.
7) export the code as runnable jar.
8) use Launch4j to convert the jar to .exe for distribution.
9) ENJOY and relax while the automation runs.
